# 宏观数据月度更新标准操作流程 (SOP)

**版本**: v1.0  
**最后更新**: 2026-05-25  
**适用范围**: Investment Agent 项目宏观数据更新  

---

## 一、流程概述

```
每月初(9-15日)                填写数据                      一键导入
     |                            |                              |
     v                            v                              v
[下载Excel] --→ [放入downloads] --→ [运行导入脚本] --→ [自动重算] --→ [查看Dashboard]
     |                            |                              |
     统计局官网                    解析+校验+入库                  因子+状态更新
     央行官网
```

**总耗时**: 约 10-15 分钟（其中手动操作约 5 分钟）

## 二、详细步骤

### Step 1: 下载数据（每月9-15日）

#### 1.1 统计局数据（PMI + CPI + PPI + 工业增加值）

**发布时间**:
- PMI: 每月最后一天或次月1日
- CPI/PPI: 每月9-11日
- 工业增加值: 每月中旬

**下载地址**:
1. 打开 https://data.stats.gov.cn/
2. 导航: 月度数据 → 选择指标
3. 选择时间范围（建议下载最近3个月，便于核对）
4. 导出为 Excel (.xlsx)

**需要下载的指标**:
| 指标 | 网页路径 | 频率 |
|------|---------|------|
| 制造业PMI | 月度数据 → PMI | 月度 |
| CPI同比/环比 | 月度数据 → 价格指数 → CPI | 月度 |
| PPI同比/环比 | 月度数据 → 价格指数 → PPI | 月度 |
| 工业增加值同比 | 月度数据 → 工业 → 工业增加值 | 月度 |

#### 1.2 央行数据（M0/M1/M2 + 社融）

**发布时间**: 每月10-15日

**下载地址**:
1. 打开 http://www.pbc.gov.cn/
2. 导航: 统计数据 → 货币统计 / 社会融资规模
3. 下载 Excel 或复制数据

**需要下载的指标**:
| 指标 | 网页路径 | 频率 |
|------|---------|------|
| M0/M1/M2同比 | 统计数据 → 货币统计概况 | 月度 |
| 社融存量同比 | 统计数据 → 社会融资规模 | 月度 |
| 社融当月值 | 统计数据 → 社会融资规模 | 月度 |

#### 1.3 放入指定目录

将下载的所有 Excel 文件放入:
```
data_external/downloads/
```

支持子目录，系统会自动递归搜索。

2. 浏览器访问 http://localhost:3011

3. 检查:
   - ✅ 截面分析页面显示最新月份: 2026年4月

#### 4.3 核对关键指标

打开截面分析页面，核对以下数值:
- PMI: 应与统计局发布一致
- CPI/PPI: 应与统计局发布一致
- M2: 应与央行发布一致
- 社融: 应与央行发布一致

### Step 5: 处理异常（如需要）

#### 场景1: 导入失败

```bash
# 查看详细错误
python scripts/data_import/macro_data_importer.py --auto
```

常见原因:
- Excel格式不识别 → 尝试另存为 CSV
- 数据超出范围 → 检查数值是否正确
- 日期格式错误 → 确认日期列为 YYYYMMDD 格式

#### 场景2: 数据需要修正

直接修改 `data_external/downloads/` 中的文件，重新运行导入。
系统会自动覆盖旧数据（UPSERT）。

#### 场景3: 手动补充单个指标

```bash
# 生成模板，填入数据，导入
python scripts/data_import/macro_data_importer.py --template --year 2026 --month 4
# ... 编辑模板 ...
python scripts/data_import/macro_data_importer.py --auto
```

## 四、常见问题

**Q1: 统计局Excel格式有变化，解析失败怎么办？**

A: 
1. 尝试另存为 CSV 格式
2. 或手动复制数据到 `macro_template_YYYYMM.csv` 模板
3. 如果问题持续，联系开发团队更新解析器

**Q2: 只想更新部分指标，不想全部重新导入？**

A: 系统采用 UPSERT 策略，只更新Excel中包含的指标，不影响其他数据。

**Q3: 导入后发现数据错误，如何回滚？**

A: 目前不支持自动回滚。建议:
1. 修正Excel中的错误数据
2. 重新运行导入脚本
3. 或手动在数据库中修正（需要SQL操作）

**Q4: OMO利率和DR007等日度数据如何更新？**

A: 日度数据建议:
1. 批量导入：每月初一并导入上月所有日度数据
2. 或保持现有更新频率（日度数据已有到2026-05-07）

**Q5: 可以设置定时自动抓取吗？**

A: 当前版本不支持自动抓取（因统计局/央行反爬限制）。
建议每月10-15日手动下载后一键导入。
