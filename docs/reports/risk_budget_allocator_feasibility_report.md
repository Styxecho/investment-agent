# 风险预算配置模型在中国市场的可行性研究报告

**报告日期**：2026-06-26  
**研究范围**：2015-01-06 至 2026-06-26（共 2,787 个交易日）  
**核心资产**：中证全指（000985.CSI）、中债-总财富(7-10年)指数（CBA00351.CS）、上海黄金（AU.SHF）  
**研究结论**：在现行约束条件下，风险预算（Risk Budgeting）+ 目标波动率（Target Volatility）方法难以在中国股债市场构造出具有显著风险梯度的大类资产配置方案。本报告系统记录研究动机、方法、实施细节、实证结果及反思。

---

## 1. 研究背景与目标

### 1.1 研究动机

在境外成熟市场中，风险平价（Risk Parity）和风险预算模型被广泛用于多资产配置。其思想是：不依赖对资产收益率的预测，仅通过风险贡献的分配来决定权重，从而获得在不同市场环境下更稳健的组合表现。我们希望在 investment-agent 框架内实现一个可独立发布的大类资产配置模块，服务于以下场景：

- 保守组合：年化波动约 3%，适合稳健型资金。
- 均衡组合：年化波动约 5%，适合平衡型资金。
- 激进组合：通过更高风险暴露追求长期收益。

### 1.2 核心约束

考虑到中国内地场外基金投资的实际环境，我们设定了以下硬约束：

1. **不依赖预期收益率**：仅使用历史波动率和协方差，避免 mean-variance 对收益估计的敏感。
2. **无杠杆**：风险资产权重之和不超过 1.0，剩余部分为现金。
3. **目标波动率上限**：组合年化波动尽量接近但不超过目标值。
4. **可配置的风险预算**：允许保守组合低配股票、激进组合高配股票。

### 1.3 研究目标

验证风险预算 + 目标波动率框架是否能够在中国股、债、商品三类资产上生成风险特征明显区分、长期表现合理的三个组合。

## 2. 方法论

### 2.1 风险预算模型

风险预算模型要求各资产的风险贡献（Risk Contribution, RC）与预设的风险预算（Risk Budget, b）成比例：

```
RC_i = w_i * (Σw)_i / (w^T Σw) ≈ b_i
```

其中：
- `w` 为资产权重向量
- `Σ` 为资产的年化协方差矩阵
- `b` 为风险预算向量，Σb_i = 1

与风险平价（Risk Parity）要求各资产风险贡献相等不同，风险预算允许投资者为不同资产设定不同的目标风险贡献比例。

### 2.2 目标波动率缩放

求解风险预算权重后，得到的基础组合波动率 `σ_rb = sqrt(w^T Σw)` 通常与目标波动率不一致。我们按以下规则缩放：

```
scale = min(target_vol / σ_rb, volatility_cap / σ_rb)
w_scaled = w_rb * scale
```

如果 `σ_rb` 低于目标波动率，则保持 `scale = 1.0`，即不主动放大风险暴露。

### 2.3 三种 Allocator 设计

| Allocator | 用途 | 核心逻辑 |
|---|---|---|
| `risk_budget` | 保守 / 均衡组合 | 风险预算求解 + 目标波动率缩放 + 权重约束 |
| `target_vol` | 可选对比 | 直接优化权重以满足目标波动率，最大化风险资产使用率 |
| `manual` | 激进组合 | 人工固定权重 70% 股 / 20% 债 / 10% 商品 |

### 2.4 协方差估计

采用 Ledoit-Wolf 收缩估计，回望窗口 252 个交易日，年化因子 252。相比样本协方差，Ledoit-Wolf 在样本量有限或资产波动差异较大时更稳定。

### 2.5 求解器链路

风险预算求解器采用四级 fallback 机制：

1. **SLSQP**：带权重边界和权重和为 1 的约束求解。
2. **trust-constr**：内点法，作为 SLSQP 失败后的兜底。
3. **无约束风险预算解 + 约束裁剪**：去除边界约束求解后再裁剪归一化。
4. **逆波动率近似**：`w_i ∝ b_i / σ_i`，最后兜底。

### 5.5 Fallback 触发情况

在 134 个调仓日中，保守和均衡组合各有 69 个有效样本，全部触发了 fallback 逻辑。这说明在全部有效调仓日，SLSQP 和 trust-constr 求解器无法在给定约束下收敛到满足风险预算的最优解，系统不得不退回到无约束解或逆波动率近似。

激进组合使用固定权重，未触发 fallback。

## 6. 关键发现与讨论

### 6.1 风险预算模型在中国市场的核心失效点

**债券波动率过低，无法支撑有意义的风险分配。**

在风险预算框架中，组合的权重由“风险预算”驱动。当债券资产的年化波动仅为 2% 而股票为 23% 时，即使给股票设定 40% 的风险预算，其对应的权重也可能只有 5-10%。模型天然会把绝大部分资金配置到债券上，以匹配设定的风险预算比例。

实证结果完全印证了这一点：
- 均衡组合的目标风险预算为 40% 股票 / 50% 债券 / 10% 商品，但实际股票权重均值仅 10.9%，债券权重均值 88.4%。
- 保守组合的目标风险预算为 15% 股票 / 80% 债券 / 5% 商品，实际股票权重均值仅 5.6%。

### 6.2 目标波动率无法实现

由于无杠杆约束，组合无法通过放大债券暴露来达到中等波动目标：
- 保守组合目标 3%，实际仅 1.4%（中位数 1.3%）。
- 均衡组合目标 5%，实际仅 2.4%（中位数 2.0%）。

若要达到 5% 目标波动，在无杠杆条件下需要大幅提高股票权重（估计需 25-30%），但这会显著偏离风险预算模型的最优解，并频繁触发权重上限约束。

### 6.3 激进组合的困境

激进组合采用固定权重 70/20/10，其长期表现反而不如保守和均衡组合：
- 年化收益 2.16%，低于保守（2.30%）和均衡（2.63%）。
- 最大回撤 -25.60%，波动率 9.57%。
- Calmar 比率仅 0.08。

这说明在 2015-2026 这段样本期内，A 股整体收益不足以补偿其高波动风险。固定高股票权重的策略在中国长期表现平庸。

### 6.4 风险预算求解器的稳定性问题

保守和均衡组合在全部有效调仓日都触发了 fallback 机制。原因包括：
- 权重边界约束与风险预算约束同时存在时，可行域可能非常狭窄。
- 债券波动率过低导致协方差矩阵条件数较大，优化问题病态。
- 目标波动率缩放后，部分组合的权重和已远低于 1.0，进一步加剧了归一化误差。

### 6.5 为什么 7-10 年期债券指数也没有解决问题

我们曾希望通过使用更长久期的债券指数来提高债券端波动。但从实证结果看：
- CBA00351.CS 波动率仅 2.31%，CBA00151.CS 仅 2.01%。
- 相比全期限指数的提升有限（CBA00601.CS 约 1.71%，CBA00101.CS 约 1.04%）。
- 相对于股票 23% 的波动，债券端的提升仍然是数量级上的差距。

因此，仅仅更换债券代理指数无法从根本上解决风险预算模型在中国市场失效的问题。

## 7. 失败原因总结

综合以上分析，我们认为风险预算 + 目标波动率方法在当前设定下不可行的主要原因有三点：

1. **资产波动结构极度不均衡**
   中国债券波动率（1-2%）与股票波动率（19-23%）相差一个数量级，导致风险预算模型天然超配债券。

2. **无杠杆约束压制了组合波动**
   无法通过加杠杆放大债券暴露，因此即使把债券配满，组合波动也难以达到 3%（保守）或 5%（均衡）的目标。

3. **A 股长期风险溢价不足**
   在 2015-2026 样本期内，中证全指年化收益仅 6.53%，但波动高达 22.84%。提高股票权重虽然能提升波动，但收益-风险比并不具有吸引力。

## 8. 反思与未来方向

### 8.1 本次研究的价值

尽管最终结论是方法不可行，但本次研究仍然具有记录价值：

- **排除了一条错误路径**：明确了纯粹的风险预算 / 风险平价框架在股债波动差距极大的中国市场难以直接应用。
- **建立了可复用的基础设施**：`risk_budget_allocator` 包、统一 allocator 入口、回测框架、文档体系都可以为后续研究复用。
- **量化了约束边界**：在无杠杆条件下，股债商品的波动结构决定了组合波动上限约为 8-11%，这为中高风险组合的设计提供了参考。

### 8.2 可能的改进方向

如果未来希望继续探索大类资产配置，可以考虑以下方向：

1. **引入杠杆或杠杆型工具**
   使用国债期货、融资融券、或杠杆 ETF 来放大债券/股票暴露，才能突破无杠杆下的波动上限。但这会带来新的监管和风险管理要求。

2. **设置最小风险资产配置**
   在风险预算求解之外，强制加入最低股票暴露约束（如保守 15%、均衡 30%），用半规则化方式提升组合波动。但这会偏离风险预算的纯粹性。

3. **换用目标波动率 Allocator 作为默认**
   `TargetVolAllocator` 的目标函数是“在满足波动约束下最大化风险资产使用率”，它天然会比 RiskBudgetAllocator 更积极地使用股票权重。后续可以专门对比两者的长期表现。

4. **扩展资产类别**
   引入 REITs、可转债、商品期货、海外权益等波动和收益特征更丰富的资产，可能改善组合的风险-收益结构。

5. **接受低波动产品定位**
   将本产品定位为“稳健型资产配置工具”，目标波动 2-4%，不再追求激进组合。这可能是最符合当前数据和市场结构的现实选择。

6. **引入收益预测或战术配置**
   放弃纯风险预算的“无收益预测”原则，结合宏观状态、估值、动量等信号进行战术超配/低配，可能显著提升组合表现。
