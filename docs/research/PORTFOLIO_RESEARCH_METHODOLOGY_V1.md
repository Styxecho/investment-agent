## 1. 研究定位与边界

### 1.1 与阶段一的关系
阶段一解决"我已经持有了什么，现在表现如何"（跟踪）。  
阶段二解决"我应该如何构建一个组合"（研究）。  
两者的衔接点：阶段二的研究报告可以输出一份 `recommended_holdings.csv`，直接导入阶段一的跟踪系统进行监控。

### 1.2 人机分工原则
- **Agent 负责**：数据获取、量化计算、回测执行、报告生成
- **人负责**：ETF 筛选与标签维护、组合构建方法的最终选择、投资决策
- **明确边界**：阶段二不直接产出交易信号，只提供决策参考

## 2. ETF 初筛规则

在进入标签分类和组合构建之前，必须先从全市场 ETF 中筛选出"可研究"的标的。初筛标准如下：

| 维度 | 标准 | 理由 |
|------|------|------|
| 规模 | 最新规模 >= 2 亿元 | 规避清盘风险，保证流动性 |
| 成交量 | 最新年日均交易量 >= 5000 万元 | 确保可交易性，降低冲击成本 |
| 跟踪精度 | 年跟踪误差 <= 5% | 排除偏离基准过大的产品 |
| 上市时间 | 上市交易年限 > 1 年 | 保证有足够历史数据用于分析 |

**维护方式**：手工维护 `data_external/reference/etf_universe.csv`，包含通过初筛的 ETF 列表及其基础信息。  
**Review 频率**：初期每年 review 一次，逐步提高频率，不高于季频。

## 3. 数据与回测框架

### 3.1 价格口径
- **优先采用复权价格**进行收益率计算
- ** fallback**：若复权数据不可得，使用原始收盘价，并在报告中注明

### 3.2 再平衡规则
再平衡不是独立的，而是**组合构建方法的附属规则**：
- 若采用**买入持有策略**，不再平衡
- 若采用**风险平价策略**，当单个资产的风险贡献偏离目标超过阈值时触发再平衡
- 若采用**定期再平衡策略**（如等权重），按日历频率（月度/季度）复位到目标权重

### 3.3 交易成本
- **为简化，暂不考虑交易磨损**（手续费、滑点）
- 未来如需纳入，默认双边费率可设为 0.03%

### 3.4 缺失数据处理
- 若某交易日某资产无数据，当日该资产权重按 0 处理，或以前一交易日收盘价填充
- 若缺失比例超过 5%，在报告中特别提示数据可靠性问题

## 4. 组合构建与分析方法

### 4.1 相关性/协方差估计（默认参数）
- **时间窗口**：不低于 1 年（252 个交易日）
- **加权方式**：优先采用加权估计（如 EWMA），而非简单等权重
- **专题说明**：协方差矩阵的高级估计方法（收缩估计、随机矩阵去噪等）需参阅专题调研文档 `COVARIANCE_ESTIMATION_SURVEY.md`

### 4.2 组合构建方法

首期支持以下方法：
1. **等权重（Equal Weight）**
2. **指定权重（User-Defined Weight）**
3. **风险平价（Risk Parity）**
4. **其他扩展方法**：待专题调研后决定是否纳入

#### 核心池构建方法（Core Portfolio Construction）

**原则**：核心池的稳健性来自资产间的**低相关性**，而非单纯选择低波动资产。通过风险平价模型，使每只资产对组合总风险的贡献相等，从而自动实现高波动资产低权重、低波动资产高权重的配置效果。

**实证验证计划**：
为验证"资产分散性 vs 资产低波动性"哪个对核心池稳健性更重要，设计以下对照实验：

**组合 A（多元化核心池）**：
- A股大盘：510300.SH 沪深300ETF华泰柏瑞
- A股中盘：510500.SH 中证500ETF南方
- A股中小盘：512100.SH 中证1000ETF南方
- A股小盘：159531.SZ 中证2000ETF南方
- A股科创：588000.SH 科创50ETF华夏
- 港股大盘：159920.SZ 恒生ETF华夏
- 港股科技：513010.SH 恒生科技ETF易方达
- 美股成长：513100.SH 纳指ETF国泰
- 美股大盘：513500.SH 标普500ETF博时
- 商品：518880.SH 黄金ETF华安
- 债券(长久期)：511260.SH 十年国债ETF国泰
- 债券(中久期)：159972.SZ 5年地方债ETF鹏华
- 债券(短久期)：511360.SH 短融ETF海富通

**组合 B（低波动核心池）**：
- A股红利：510880.SH 红利ETF华泰柏瑞
- A股红利低波：512890.SH 红利低波ETF华泰柏瑞
- A股自由现金流：159201.SZ 自由现金流ETF华夏
- A股央企红利：561580.SH 央企红利ETF华泰柏瑞
- 港股红利：513920.SH 港股通央企红利ETF华安
- 港股红利低波：159545.SZ 恒生红利低波ETF易方达
- 债券(长久期)：511260.SH 十年国债ETF国泰
- 债券(中久期)：159972.SZ 5年地方债ETF鹏华
- 债券(短久期)：511360.SH 短融ETF海富通
- 商品：518880.SH 黄金ETF华安

**回测参数（统一控制变量）**：
- 回测区间：2019-01-01 至 2025-04-14（覆盖完整牛熊周期）
- 组合构建方法：风险平价（Risk Parity）
- 协方差估计：252日滚动样本协方差
- 再平衡频率：月度再平衡
- 交易成本：暂不考虑

**对比指标**：年化收益率、年化波动率、最大回撤、夏普比率、卡玛比率、月度正收益比例、净值曲线形态。

**核心假设**：在相同的风险平价模型下，组合 A（多元化）的年化波动率和最大回撤将低于组合 B（低波资产集中），且夏普比率更高。

#### 核心-卫星仓位分配
- **核心池**：80% 仓位
- **卫星池**：20% 仓位
- 卫星池用于行业轮动、风格择时、主题热点等增强策略
- 未来可通过风险预算法或动态再平衡优化该比例

### 4.3 分析对象
- 同时支持 **ETF** 和 **个股**，BacktestSkill 本身不区分资产类型
- 阶段二以 ETF 为主场景，但代码层保持通用

## 5. 绩效指标口径

### 5.1 必选指标
| 指标 | 说明 |
|------|------|
| 累计收益率 | 区间总收益 |
| 年化收益率 | 按交易日年化 |
| 年化波动率 | 收益率标准差，年化 |
| 最大回撤 | 区间内的最大峰值到谷底跌幅 |
| 卡玛比率 | 年化收益率 / 最大回撤 |
| 年化换手率 | 再平衡场景下，年度调仓比例 |

### 5.2 基准
- **阶段二只看绝对收益**，不设置基准对比
- 未来如需相对收益分析，可再引入沪深 300 或自定义复合基准

## 6. 研究报告规范

### 6.1 触发方式
用户通过自然语言触发，典型问法包括：
- "帮我分析一下股债 3:7 的组合过去三年的表现"
- "如果我现在买入科技 ETF 和医药 ETF 各 50%，回测结果如何？"
- "请给我一份基于大类资产的均衡组合配置建议"

### 6.2 报告内容
报告为**定制化生成**，不套用固定模板，但通常包含以下模块：
- 组合构建逻辑说明（权重来源、再平衡规则）
- 回测绩效指标表
- 净值走势的文字描述
- 各资产贡献拆分（市值、收益贡献）
- 风险提示与数据局限性说明

### 6.3 输出形式
- 默认输出 Markdown 格式的自然语言报告
- 可选输出 `recommended_holdings.csv`，直接对接阶段一的 PortfolioSkill
